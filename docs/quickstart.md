# Quickstart

!!! note
    This page is a placeholder. The complete quickstart (install, configure, index, ask, drift)
    is written in Step 10.

## Install

```bash
pip install repolens
```

## Index a repository

```bash
repolens index ./path/to/repo
```

## Ask a question

```bash
repolens ask "how does request routing work?" --repo <id>
```

## Run a drift check

```bash
repolens drift --repo <id>
```

## Launch the web UI

```bash
repolens serve
# open http://localhost:8000
```
